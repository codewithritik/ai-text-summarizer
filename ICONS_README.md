# Icon Files Required

The extension manifest references icon files that need to be created:

- `icon16.png` (16x16 pixels)
- `icon48.png` (48x48 pixels)  
- `icon128.png` (128x128 pixels)

## Quick Solution

You can:
1. Create simple icons using any image editor
2. Use online icon generators
3. Download free icons from sites like:
   - [Flaticon](https://www.flaticon.com)
   - [Icons8](https://icons8.com)
   - [Font Awesome](https://fontawesome.com)

## Temporary Workaround

If you want to test the extension without icons, you can temporarily comment out the icon references in `manifest.json`, but Chrome will show a default icon.

The extension will work without icons, but Chrome may show warnings during installation.
